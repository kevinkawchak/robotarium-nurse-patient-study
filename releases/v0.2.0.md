Release title
v0.2.0 -

## Summary
This release introduces a new, improved Robotarium-compatible Python simulator that runs both locally and in-browser via GitHub Pages.

## Features
- Robotarium API compatibility layer (`rps`) with safety and motion primitives.
- Browser simulator for simple `.py` upload + play interaction.
- Default script support for `Exp_01a_12Feb26.py`.
- CI lint/format checks targeting Python 3.10, 3.11, and 3.12.
