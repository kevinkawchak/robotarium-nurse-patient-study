## Overview

This demo was run using the Robotarium's built-in demonstration feature, which showcases the platform's swarm robotics capabilities.

## Description

The demo features multiple GRITSBot robots coordinating to form patterns, demonstrating:
- Multi-robot coordination
- Formation control algorithms
- Barrier certificate collision avoidance
- Autonomous wireless charging return

## About the Robotarium Demo Page

The Robotarium provides pre-built demonstrations that users can run to see the platform in action before creating custom experiments. These demos showcase various swarm robotics capabilities including formation control, consensus algorithms, and coordinated motion.
